# Pasta /data (runtime)

Esta pasta é usada **apenas em execução**.

Arquivos reais (gerados):
- `saida_ops.json` (operações cadastradas)
- `saida_monitor.json` (tabela pronta para o painel)
- `saida_worker_err.log` (erros do worker)

⚠️ Estes arquivos **não** devem ir para o GitHub. Veja `.gitignore`.

Modelos de exemplo ficam em `data/examples/`.
